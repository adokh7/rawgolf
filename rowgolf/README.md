# Rawgolf

Rawgolf is a media and news platform dedicated to golf, providing unfiltered, honest coverage of the sport. It features data-driven player ratings, tactical tournament analysis, swing breakdowns, and golf culture stories. 

## Purpose
Our goal is to cover golf the way it deserves: honest news, no fluff, and no PR rewrites. Every number earned, every rating final.

## Tech Stack
- HTML5
- Tailwind CSS / Custom CSS
- Vanilla JavaScript

## Project Structure
- `index.html`: The main homepage (formerly rawgolf.html).
- `article-template.html`: Template for individual news articles and analysis.
- `ratings-manual.html`: The methodology behind our Raw Player Ratings.
- `manifesto.html`: Our mission and commitment to honest journalism.
- `corrections.html`: A public log of corrections for accountability.
- `privacy.html`: Privacy policy and cookie consent details.
- `contact.html`: Contact information.

## Deployment
